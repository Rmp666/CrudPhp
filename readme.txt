Требования:

    Composer 1.7.2
    PHP 7.0
    MySQL 5.6 
    Node 8.9.1

Запуск:

    1) Composer install

    2) composer dump-autoload

    3) npm install

    4) nmp run web

    5) Изменить название освновной директории проекта в config/app.php и .htaccess

    6) Создать бд

    7) Изменить название бд в файле config/DB.php

    8) Открыть при первом запуске URL: Домен/название освновной директории/models/Launch.php
